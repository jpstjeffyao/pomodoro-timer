# 平靜番茄鐘 | Calm Pomodoro

一個兼具美感與實用性的生產力工具，幫助您在專注與休息之間取得平衡。

## 專案目的 (Project Purpose)

「平靜番茄鐘」的開發初衷是為使用者提供一個無壓力、視覺舒適的專注環境。在快節奏的工作中，番茄鐘工作法（Pomodoro Technique）能有效提升效率，而這款應用程式透過柔和的色彩與簡約的介面，旨在減少使用者的焦慮感，讓您能更平靜地進入專注狀態。

## 設計過程 (Design Process)

在設計此應用程式時，我們遵循了以下核心原則：

1.  **視覺美感 (Aesthetics)**：
    *   **色調選擇**：使用了柔和的鼠尾草綠（#d4e0d4）代表專注模式，以及暖沙色（#e0d4d4）代表休息模式，營造出和諧且自然的氛圍。
    *   **玻璃擬物化 (Glassmorphism)**：主介面採用半透明、模糊背景的設計，增加層次感與現代感。
    *   **動態效果**：加入了細微的背景漸變旋轉動畫，讓畫面顯得生機勃勃但不干擾使用者專注。

2.  **使用者體驗 (UX)**：
    *   **簡潔操作**：將常用的控制按鈕（開始、暫停、重置）置於中心。
    *   **雙重功能**：除了計時器，還整合了簡單的待辦事項清單，落實「一個視窗處理專注與任務」的需求。
    *   **即時反饋**：網頁標題會動態顯示剩餘時間，方便使用者在切換頁面時仍能掌握進度。

## 主要功能 (Features)

*   **專業番茄鐘**：內建 25 分鐘專注模式與 5 分鐘休息模式。
*   **自動切換**：當時間歸零時會自動切換至另一模式。
*   **提示音效**：計時結束時會播放柔和的正弦波提示音，提醒您該休息或開始工作。
*   **生產力清單**：支援新增、勾選完成以及刪除待辦事項。
*   **響應式設計**：支援各種螢幕尺寸，從電腦到手機都能完美顯示。

## 使用方式 (Usage)

1.  **開啟應用程式**：
    直接使用瀏覽器開啟專案資料夾中的 `index.html` 檔案即可。
2.  **啟動計時器**：
    點擊「開始」按鈕啟動倒數。您可以在上方切換「專注」或「休息」模式。
3.  **管理任務**：
    在下方的輸入框中輸入任務內容，按下「+」按鈕或 `Enter` 鍵即可新增至清單。
4.  **完成與刪除**：
    勾選任務左側的核取方塊可標記為完成；將滑鼠移至任務上方點擊右側的「×」可刪除該任務。

## 技術堆棧 (Tech Stack)

*   **結構**：HTML5
*   **樣式**：Vanilla CSS3 (含 Flexbox, Grid, 關鍵幀動畫)
*   **邏輯**：Vanilla JavaScript (ES6+)
*   **字體**：Google Fonts (Quicksand)
